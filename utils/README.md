# Utility Scripts

This folder contains diagnostic scripts and analysis tools for troubleshooting and exploring citation data.

## Diagnostic Scripts

- **verify_cache.py** - Check the contents and format of cached affiliation data
- **check_duplicates.py** - Detect duplicate entries in affiliation cache
- **check_uniqueness.py** - Analyze uniqueness of author entries
- **analyze_matching.py** - Analyze how affiliations match to author indices

## Notebooks

- **analyze_partial_affiliations.ipynb** - Analyze partial affiliation data during scraping (useful for checking progress mid-run)
